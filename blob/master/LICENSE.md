Scope of Functionality

ScoutHawkLite is a macOS application developed in Swift that provides the following
features, operating entirely on the local system:

Real-time monitoring of system resources, including CPU and memory usage

Monitoring of file and folder changes, including creation, deletion,
renaming, and modification of user-selected directories and files

Monitoring and status tracking of macOS Privacy & Security permissions,
including Accessibility, Screen Recording, Input Monitoring, and
Full Disk Access

Menu bar user interface for system visibility and interaction

Local system notifications related to monitored events

Privacy & Data Handling

ScoutHawkLite does not collect, store, transmit, or share personal data.
All monitoring, detection, and notification functionality is performed
locally on the users device.

Any macOS system permissions requested by ScoutHawkLite are required solely to enable
explicitly described features and are granted voluntarily by the user.
ScoutHawkLite does not bypass, override, or weaken macOS security mechanisms.
